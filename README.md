# Site-web-evenement-Tresors-de-Banlieue
Projet Universitaire effectué en groupe avec des amis de la formation. Le but était de créer un site web avec mon groupe de projet pour un évènement qui aurait lieu.
Nous avons ainsi dû choisir un évènement public ayant lieu, contacter l'organisateur de l'évènement afin de lui expliquer notre projet et nous organiser afin de visiter l'évènement dans le but de se renseigner, prendre des photos de l'évènement, récupérer des avis des personnes présentes, et faire le site web pour l'évènement à partir de tous ce qui à été fait.
